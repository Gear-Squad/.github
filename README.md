# 🎯 Sobre Nós

A **Gear Squad** é uma equipe acadêmica formada por estudantes dos cursos de **Análise e Desenvolvimento de Sistemas** e **Ciência da Computação** da **FASEH**.

Nossa proposta é transformar os conhecimentos adquiridos em sala de aula em projetos práticos, desenvolvendo soluções que permitam aplicar conceitos de programação, lógica computacional, engenharia de software e trabalho colaborativo.

Buscamos evoluir continuamente por meio da construção de sistemas reais, utilização de boas práticas de desenvolvimento e compartilhamento de conhecimento entre os integrantes da equipe.

# 📂 Projetos

## 📦 Gear Squad — Sistema de Controle de Estoque

Sistema desenvolvido em Java para gerenciamento de estoque via terminal.

### Funcionalidades

✅ Cadastro de Produtos

✅ Entrada de Estoque

✅ Saída de Estoque

✅ Busca de Produtos

✅ Edição de Registros

✅ Listagem de Produtos

✅ Exclusão de Produtos

✅ Limpeza Completa do Estoque

✅ Validações de Dados

---

## 🔄 Fluxo Geral do Sistema

```text
INÍCIO
   │
   ▼
MENU PRINCIPAL
   │
   ├── Cadastro
   │
   ├── Movimentação
   │      ├── Entrada
   │      └── Saída
   │
   ├── Listagem
   │
   ├── Edição
   │
   ├── Exclusão
   │
   ├── Limpar Lista
   │
   └── Sair
   │
   ▼
FIM
```

# 🧠 Estrutura do Projeto

```text
src/
├── Main.java
│
└── Classes/
    ├── Produto.java
    └── EstoqueService.java
```

### Main

Responsável pela interface e navegação do sistema.

### Produto

Responsável pela modelagem dos produtos cadastrados.

### EstoqueService

Responsável pelas regras de negócio e gerenciamento do estoque.
